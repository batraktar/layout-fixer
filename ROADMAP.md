# Roadmap

## v0.1 — Core MVP

- Background app with tray/menu bar icon
- Global shortcut (`Cmd+Shift+L` / `Ctrl+Shift+L`)
- Clipboard-based text replacement
- English ↔ Ukrainian layout conversion
- JSON-backed layout architecture
- Basic debug/settings UI

## v0.2 — Product polish

- Better first-launch onboarding
- macOS Accessibility permission guide
- Improved clipboard restore reliability
- Demo GIF in README
- Better error messages

## v0.3 — Releases

- macOS DMG installer
- Windows NSIS installer
- Linux AppImage / .deb
- GitHub Actions CI/CD
- GitHub Releases with auto-generated installers

## v0.4 — More layouts

- More built-in language pairs
- Enable/disable individual layouts from UI
- Custom layout file support
- Layout contribution guide

## v0.5 — Retro UI

- Windows 95/98 inspired design
- Pixel-style tray icon
- Classic system-font aesthetic
- Tray and menu polish
